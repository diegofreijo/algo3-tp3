\subsection{Conclusiones}
Las conclusiones son bastante claras. El algoritmo se comporto como se esperaba, tardando mucho en grafos medianamente chico para ejecutar. Se ve claramente como el algoritmo no es recomendable para resolver este problema, aunque se quiera una solucion precisa, ya que demoraria meses, años o siglos en tener la misma. En la carpeta bib se adjunta un pdf con una solucion que baja la complejidad a $O(1.87^{n})$ aproximadamente, aunque en grafos muy especificos.
