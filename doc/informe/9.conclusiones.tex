\section{Conclusiones}
Luego de haber analizado a cada algoritmo por separado y luego analizarlo juntos, se llega a las siguientes conclusiones:

\begin{itemize}
\item El goloso es rapido, mas comparado al GRASP, pero sus soluciones siempre estan dentro de las peores comparando el tamaño.
\item La BL, tiene un tiempo de ejecucion medio entre Goloso y GRASP, y sus soluciones son aproximadas al GRASP.
\item GRASP tiene los mejores resultados, pero es costoso si se mira el tiempo de ejecucion.
\end{itemize}

Con estas conclusiones se puede advertir que:

\begin{itemize}
\item Para grafos chicos, se recomienda GRASP o BL, ya que su solucion es aproximada a la exacta, y al ser poca la cantidad de nodos, el tiempo de ejecucion no influye demasiado.
\item Para grafos grandes, Goloso o BL, el primero en los casos en que se quiera rapidez y menos eficacia y viceversa para el segundo.
\end{itemize}
